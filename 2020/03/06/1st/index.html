安装准备

需要安装node.js和git两部分,找到对应系统的版本安装即可

  • nodejs

  • git

安装Hexo

使用npm(一种包管理器，和python的pip，conda，centos使用的yum等类似)


npm install hexo-cli -g

如果你的网络质量不好，可以使用镜像


npm install hexo-cli -g --registry=镜像源

可用镜像源有：

名称|URL|

-|-|-

淘宝NPM镜像|https://registry.npm.taobao.org

华为云镜像|https://mirrors.huaweicloud.com/repository/npm/

然后执行以下命令来初始化


hexo init myBlog

这一条要会使用git来克隆相关文件到你本地的，记得挂上魔法上网工具，否则很慢很慢很慢。

然后切换到myBlog目录下，使用npm install命令来自动地根据package.json文件来安装需要的依赖（有些依赖文件可能很大，执行的时候再安装比较轻量）


cd myBlog

npm install

网上所说的安装好目录是这样的


.

├── _config.yml # 网站的配置信息，您可以在此配置大部分的参数。

├── package.json

├── scaffolds # 模版文件夹

├── source  # 资源文件夹，除 _posts 文件，其他以下划线_开头的文件或者文件夹不会被编译打包到public文件夹

| ├── _drafts # 草稿文件

| └── _posts # 文章Markdowm文件

└── themes # 主题文件夹

而我自己安装好是这样的，不清楚有什么区别


.

├── _config.yml # 网站的配置信息，您可以在此配置大部分的参数。

├── package.json # 依赖项配置文件

├── package-lock.json

├── scaffolds # 模版文件夹

├── node_modules # hexo的依赖项

├── .gitignore

├── source  # 资源文件夹，除 _posts 文件，其他以下划线_开头的文件或者文件夹不会被编译打包到public文件夹

| └── _posts # 文章Markdowm文件

└── themes # 主题文件夹

然后输入以下命令来启动本地服务器


hexo server

如果你和我一样用windows，会弹出相应的防火墙，点击允许即可。

然后我们在浏览器输入http://localhost:4000 就可以看到你的博客了（本地）
